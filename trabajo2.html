CONCEPTO:
Las tecnologias Web se refiere a las aplicaciones que lo usuarios pueden utilizar accediendo a un servidor web a traves de internet o de una intranet mediante un navegador. Es una aplicacion de software que se codifica en un lenguaje soportado por los navegadores web en la que se confia la ejecucion al navegador.



FUNCIONAMIENTO:
Generan dinamicamente una serie de paginas en un formato estandar, como HTML o XHTML, soportados por los navegadores web comunes. Se utilizan lenguajes interpretados en el lado del cliente, directamente o a traves de plugins tales como JavaScript, Java, Flash, etc., para a√±adir elemtos dinamicos ala interfaz de usuario. Generalmente cada pagina web en particular se envia al cliente como un documento estatico, pero la secuencia de paginas ofrece al usuario una experiencia interactiva. Durante la sesion, el navegador web interpreta y muestra en pantalla las paginas, actuando como cliente para cualquier aplicacion web.



CLIENTE:
Existen varios navegadores web que hacen peticiones a servidores para obtener una pagina de internet solicitada por un usuario. Por ejemplo:

  1. Internet Explorer
  2. Google Chrome
  3. Mozilla Firefox
  4. Safari

Asi como existen navegadores, tambien existen tecnologias de programacion para la creacion de estas paginas acontinuacion veremos unos ejemplos:>

  1. HTML
  2. JavaScript / Jscript
  3. Applets Java
  4. Componentes ActiveX en Visual C++, Visual Basic



SERVIDOR:
Hablemos de servidores, existe una gran variedad de servidores que usan muchas empresas para guardar sus documentos html y asi poder manejar sus paginas con mas eficacia y eficiencia. Es el programa que, utilizando el protocolo de comunicaciones HTTP, es capaz de recibir peticiones de informacion de un programa cliente (navegador), recuperar la informacion solicitada y enviarla al programa cliente para su visualizacion por el usuario. Por ejemplo:

  1. Internet Information Server (IIS)
  2. Apache, Apache-Tomcat
  3. WebSphere webserver
  4. Motores Java



Esta en un breve representacion de como funciona

El cliente le manda una peticion al servidor, el servidor la procesa, y finalmente este manda una respuesta al cliente. Todo esto bajo el protocolo HTTP.
